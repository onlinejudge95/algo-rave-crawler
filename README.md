algorave-crawler
================

## Info
This crawler collects data from [Algo rave](https://algorave.com/).

In order to run the script, you need a `Python-3.7.3` virtual environment, and all the dependencies installed.
You can create one using `bin/make_env.sh`.

## Dependencies
Project dependencies are listed in `requirements.txt`.

Python version used is `3.7.3`.
